We are constantly tweaking to this site. So there should be a way for users to know what has changed without having to click through every link on every visit. This page is that way.

This page will summarize any changes to content, and as long as your browser has cookies enabled, you'll see a little number next to "Updates" on the main screen letting you know that we've tweaked something.

An earlier version of this site used to live at DavidColarusso.com, it has since moved to a CPCS GitHub page where it underwent a major overhaul and relaunch with contributions from CPCS legal-tech interns Jodie Gurry and Vedika Mehera.
